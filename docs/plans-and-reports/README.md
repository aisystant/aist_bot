# Планы и отчёты

> Папка для хранения отчётов о прогрессе и планов развития проекта.

---

## Структура

| Тип файла | Формат имени | Описание |
|-----------|--------------|----------|
| Отчёт о прогрессе | `YYYY-MM-DD-progress.md` | Что сделано за период |
| План | `YYYY-MM-DD-plan.md` | Планы на ближайший период |
| Ретро | `YYYY-MM-DD-retro.md` | Анализ прошедшего периода |

---

## Шаблон отчёта о прогрессе

```markdown
# Отчёт о прогрессе за [период]

## Ветка main (основной бот)
- Пункт 1
- Пункт 2

## Ветка new-architecture (новый бот)
- Пункт 1
- Пункт 2

## Что дальше
- Планы
```
